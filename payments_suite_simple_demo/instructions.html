Simple Adonis + Playwright Example

1. Start the server

cd payments-suite
node ace serve --watch
Open browser: 
http://127.0.0.1:3333

2. Example Endpoint

Endpoint URL: 
POST /payments/calcular
Example request: 
curl -X POST http://127.0.0.1:3333/payments/calcular \
-H "Content-Type: application/json" \
-d '{"salary":1000000,"disability":20}'

3. Install Playwright

npm install -D @playwright/test
npx playwright install

4. Run tests

npx playwright test

5. View report

npx playwright show-report

This minimal project contains:

  • Example endpoint controller
  • Route configuration
  • Playwright API test

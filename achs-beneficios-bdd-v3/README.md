# ACHS – Motor de Beneficios (BDD v3)

Proyecto BDD avanzado alineado a ISTQB.

## Incluye:
- Cobertura completa de límites
- Cobertura total tabla de factores
- Validación financiera con Decimal
- Validación contrato JSON con Pydantic
- Separación clara de responsabilidades
- Arquitectura extensible

## Ejecución:
pip install -r requirements.txt
